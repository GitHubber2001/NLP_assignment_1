# NLP_assignment_1
This is the repository for the first NLP assignment.

We trained a Logictic Regression and a Linear SVM model on the AG News using using word-level TF-IDF features to classify documents into either 1: World 2: Sports 3: Business 4: Sci/Tech.
We also did error analysis using various evaluation matrics.

# Dataset
AG News:
- Link: https://huggingface.co/datasets/sh0416/ag_news
- Version: Version 3, Updated 09/09/2015
